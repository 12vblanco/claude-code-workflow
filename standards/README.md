# Standards

Per-language coding standards and a high-level architecture diagram.

Each language lives in its own folder with two files:

- `coding-standards.md` — conventions based on best practice, industry standards, and
  official docs
- `architecture.md` — a high-level reference architecture (Mermaid diagram)

Seeded with `javascript-typescript/`, `react/`, and `vue/`. To add a language, copy the same
two files into a new `standards/<language>/` folder.
